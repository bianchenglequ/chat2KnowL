# 简介
知识文档问答，用户大模型与文档对话，提供Al分析、阅读、问答工具，助你快速了解文档内容。

相关教程以及文档，优先发布在微信公众号 **编程乐趣**，欢迎关注！

# 项目计划
1、文件上传或网址输入：支持PDF、Doc；输入文件地址，自动下载；

2、数据库：支持连接数据库，并基于数据库的数据问答；

3、对话式问答：提供Al分析、阅读、问答工具，助你快速了解文档内容；

4、更多...
